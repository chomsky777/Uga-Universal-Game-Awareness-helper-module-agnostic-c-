The UGA Kernel exists to provide a compact, evolving, expressive intelligence layer that can be embedded into any game engine or genre.

It transforms:

context → meaning

meaning → intention

intention → expression

experience → evolution

It is the smallest possible system that still behaves like a living, adaptive micro‑LLM.

VI. Kernel Guarantees
The UGA Kernel guarantees:

1. Universality
Works in any genre, engine, or platform.

2. Determinism (optional)
Reproducible behavior when evolution is disabled.

3. Adaptability

4. φ‑Driven Coherence

VIII. Kernel Invariants
These rules must never be violated:

Perception always runs before Decision.

Decision always runs before Expression.

Adaptation always runs after Expression.

φ‑modulation is always applied.

Evolution is always bounded.

The kernel never performs game actions directly.

The kernel only outputs intent, parameters, and text.

All customization is external to the kernel.

These invariants define the kernel’s identity.
